Workflow¶

Workflow functions for assessing mutual information and statistics.

The following functions are pipelines designed for computing the mutual information and statistics (fixed effect or random effect).

WorkflowMiStats([mi_type, inference, verbose])

Workflow of mutual-information and statistics.

Dataset¶

Dataset’s related functions and classes.

This submodule contains functions and classes to handle electrophysiological data coming from several subjects.

DatasetEphy(x, y, roi[, z, times, …])

Dataset of electrophysiological data coming from several subjects.

Statistics¶

Fixed and random effect methods.

Fixed effect (ffx)¶

ffx_maxstat(mi, mi_p[, alpha])

Maximum statistics correction for fixed effect inference.

ffx_fdr(mi, mi_p[, alpha])

FDR correction for fixed effect inferences.

ffx_bonferroni(mi, mi_p[, alpha])

Bonferroni correction for fixed effect inferences.

ffx_cluster_maxstat(mi, mi_p[, alpha])

Maximum statistics for fixed effect and cluster level inferences.

ffx_cluster_fdr(mi, mi_p[, alpha])

FDR correction for fixed effect and cluster level inferences.

ffx_cluster_bonferroni(mi, mi_p[, alpha])

Bonferroni correction for fixed effect and cluster level inferences.

ffx_cluster_tfce(mi, mi_p[, start, step])

TFCE correction for fixed effect and cluster level inferences.

Random effect (rfx)¶

rfx_cluster_ttest(mi, mi_p[, alpha, center, …])

T-test across subjects for random effect inference.

rfx_cluster_ttest_tfce(mi, mi_p[, alpha, …])

TFCE and T-test across subjects for random effect inference.

Cluster¶

find_temporal_clusters(mi, mi_p, th[, tail])

Find clusters in the array of true mi.

Core¶

Core functions of information theoretical measures.

This submodule contains all of the functions to compute mutual-information, entropy etc..

For further details about the Gaussian-Copula Mutual information see [5]

Gaussian-Copula (1d)¶

Gaussian-Copula based measures to apply to unidimensional vectors

copnorm_1d(x)

Copula normalization for a single vector.

copnorm_cat_1d(x, y)

Categorical Copula normalization for a single vector.

ent_1d_g(x[, biascorrect])

Entropy of a Gaussian variable in bits.

mi_1d_gg(x, y[, biascorrect, demeaned])

Mutual information (MI) between two Gaussian variables in bits.

mi_model_1d_gd(x, y[, biascorrect, demeaned])

Mutual information between a Gaussian and a discrete variable in bits.

mi_mixture_1d_gd(x, y)

Mutual information between a Gaussian and a discrete variable in bits.

cmi_1d_ggg(x, y, z[, biascorrect, demeaned])

Conditional MI between two Gaussian variables conditioned on a third.

gcmi_1d_cc(x, y[, verbose])

Gaussian-Copula MI between two continuous variables.

gcmi_model_1d_cd(x, y[, verbose])

Gaussian-Copula MI between a continuous and a discrete variable.

gcmi_mixture_1d_cd(x, y[, verbose])

Gaussian-Copula MI between a continuous and a discrete variable.

gccmi_1d_ccc(x, y, z[, verbose])

Gaussian-Copula CMI between three continuous variables.

gccmi_1d_ccd(x, y, z[, verbose])

GCCMI between 2 continuous variables conditioned on a discrete variable.

copnorm_cat_nd(x, y[, axis])

mi_nd_gg(x, y[, mvaxis, traxis, …])

mi_model_nd_gd(x, y[, mvaxis, traxis, …])

cmi_nd_ggg(x, y, z[, mvaxis, traxis, …])

Multi-dimentional MI between three Gaussian variables in bits.

gcmi_nd_cc(x, y[, mvaxis, traxis, …])

GCMI between two continuous variables.

gcmi_model_nd_cd(x, y[, mvaxis, traxis, …])

GCMI between a continuous and discret variables.

gccmi_nd_ccnd(x, y, \*z[, mvaxis, traxis, …])

gccmi_model_nd_cdnd(x, y, \*z[, mvaxis, …])

Conditional GCMI between a continuous and a discret variable.

gccmi_nd_ccc(x, y, z[, mvaxis, traxis, …])

GCCMI between two continuous variables conditioned on a third.
