Dataset¶

Dataset’s related functions and classes.

This submodule contains functions and classes to handle electrophysiological data coming from several subjects.

DatasetEphy(x[, y, roi, z, times, …])

Dataset of electrophysiological data coming from several subjects.

Workflow¶

Workflow functions for assessing mutual information and statistics.

The following functions are pipelines designed for computing the mutual information and statistics (fixed effect or random effect).

WfMi([mi_type, inference, mi_method, …])

Workflow of local mutual-information and statistics.

WfConn([inference, mi_method, kernel, verbose])

Workflow of instantaneous pairwise connectivity and statistics.

WfFit([mi_type, inference, mi_method, …])

Workflow for Feature Specific Information Transfer.

WfStatsEphy([verbose])

Workflow of non-parametric statistics for electropĥysiological data.

Connectivity¶

Information-theritical measures of connectivity.

conn_dfc(data, times, roi, win_sample[, verbose])

Compute the Dynamic Functional Connectivity using the GCMI.

conn_covgc(data, dt, lag, t0[, step, roi, …])

Single-trial covariance-based Granger Causality for gaussian variables.

conn_fit

Compute Feature-specific Information Transfer (FIT).

conn_transfer_entropy(x[, max_delay, pairs, …])

Compute the transfer entropy.

Statistics¶

Fixed and random effect methods.

Correction for multiple comparisons¶

cluster_correction_mcp(x, x_p, th[, tail])

Cluster-based correction for MCP using non-parametric statistics.

testwise_correction_mcp(x, x_p[, tail, mcp])

Test-wise correction for MCP using non-parametric statistics.

cluster_threshold(x, x_p[, alpha, tail, …])

Threshold detection for cluster-based inferencse.

Random effect (rfx)¶

ttest_1samp(x, pop_mean[, axis, method])

One-sample t-test.

rfx_ttest(mi, mi_p[, center, zscore, ttested])

Perform the t-test across subjects.

Core¶

Core functions of information theoretical measures.

This submodule contains all of the functions to compute mutual-information, entropy etc..

For further details about the Gaussian-Copula Mutual information see [7]

Gaussian-Copula (1d)¶

Gaussian-Copula based measures to apply to unidimensional vectors

copnorm_1d(x)

Copula normalization for a single vector.

copnorm_cat_1d(x, y)

Categorical Copula normalization for a single vector.

ent_1d_g(x[, biascorrect])

Entropy of a Gaussian variable in bits.

mi_1d_gg(x, y[, biascorrect, demeaned])

Mutual information (MI) between two Gaussian variables in bits.

mi_model_1d_gd(x, y[, biascorrect, demeaned])

Mutual information between a Gaussian and a discrete variable in bits.

mi_mixture_1d_gd(x, y)

Mutual information between a Gaussian and a discrete variable in bits.

cmi_1d_ggg(x, y, z[, biascorrect, demeaned])

Conditional MI between two Gaussian variables conditioned on a third.

gcmi_1d_cc(x, y[, verbose])

Gaussian-Copula MI between two continuous variables.

gcmi_model_1d_cd(x, y[, verbose])

Gaussian-Copula MI between a continuous and a discrete variable.

gcmi_mixture_1d_cd(x, y[, verbose])

Gaussian-Copula MI between a continuous and a discrete variable.

gccmi_1d_ccc(x, y, z[, verbose])

Gaussian-Copula CMI between three continuous variables.

gccmi_1d_ccd(x, y, z[, verbose])

GCCMI between 2 continuous variables conditioned on a discrete variable.
